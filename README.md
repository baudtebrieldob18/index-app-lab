# index-app-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立开发的 HTML 页面，涵盖工具、演示、实验性功能等。每个页面均为自包含的单个 HTML 文件，不依赖外部资源（部分页面可能引用公共 CDN 库）。适合快速浏览、测试或作为代码片段参考。

## 目录结构

```
.
├── index.html          # 仓库主页 / 页面索引
├── pages/              # 归档的 HTML 页面文件夹
│   └── ...
├── assets/             # 公共资源（样式、脚本等，如有）
└── README.md           # 本文件
```

每个独立页面通常存放在 `pages/` 目录下，以 `页面名称.html` 命名。部分页面可能附带同名字文件夹存放额外资源。

## 页面归档说明

- 所有页面均为纯前端实现，不涉及后端服务。
- 页面功能、用途及注意事项请在各 HTML 文件内部的注释中查看。
- 部分页面可能为早期实验版本，代码风格或稳定性未作严格优化。
- 页面内容可能随仓库维护而更新或移除。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `index.html` 查看页面索引。
3. 或直接打开 `pages/` 下的任意 HTML 文件在浏览器中运行。

无需额外配置或构建步骤。
